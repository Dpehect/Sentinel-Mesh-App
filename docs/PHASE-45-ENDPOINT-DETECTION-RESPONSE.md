# Phase 45 — Endpoint Detection and Response

Adds deterministic endpoint behavior analysis.

Included:
- process-tree analysis
- credential dumping detection
- obfuscated PowerShell detection
- security-control tampering detection
- registry run-key persistence
- service and scheduled-task persistence
- recovery deletion and ransomware behavior
- unsigned execution from writable directories
- MITRE ATT&CK technique mapping
- process termination and endpoint isolation decisions
- unit tests
- root verification integration

Endpoint telemetry can be supplied as local JSON or imported sensor output.
No endpoint agent subscription, API key or paid service is required.
