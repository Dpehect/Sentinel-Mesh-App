# Phase 65 — Agent Health Scoring

Adds deterministic agent and fleet health scoring.

Included:
- telemetry integrity score
- runtime attestation score
- certificate health score
- policy and update compliance
- resource and connectivity health
- self-protection health
- weighted scoring
- critical finding penalties
- remediation recommendations
- healthy, investigate and contain decisions
- unit tests
- root verification integration

No external analytics or paid service is required.
