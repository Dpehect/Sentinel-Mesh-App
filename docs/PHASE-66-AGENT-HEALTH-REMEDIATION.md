# Phase 66 — Agent Health Remediation

Adds deterministic health-to-remediation orchestration.

Included:
- health recommendation mapping
- safe defensive action allowlist
- per-agent action limits
- remediation cooldowns
- priority-based planning
- human approval for network isolation
- critical compromise containment
- healthy-agent skipping
- none, remediate and contain decisions
- unit tests
- root verification integration

Arbitrary shell commands are intentionally unsupported.
No external orchestration or paid service is required.
