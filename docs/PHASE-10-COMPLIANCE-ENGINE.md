# Phase 10 — Compliance Mapping Engine

Adds deterministic compliance evidence mapping.

Included:
- SOC 2 control mapping
- ISO 27001 control mapping
- NIST CSF control mapping
- OWASP ASVS control mapping
- evidence-to-control linkage
- coverage percentage
- missing-control detection
- compact audit summary
- unit tests
- root verification integration

This is a technical mapping foundation, not a legal certification service.
No API key, subscription or paid service is required.
