# Sentinel Mesh Delivery Plan

## Phase 1 — Product foundation (included)
- Professional monorepo and independent security core
- Dynamic WebGL/Framer Motion command center
- Repository URL scan experience
- Normalized finding schema
- Contextual risk engine
- Attack path model
- SEO metadata, sitemap and robots
- Unit tests, CI and Docker baseline

## Phase 2 — Real local scanner pipeline
- PostgreSQL schema and migrations
- Redis/Valkey + BullMQ queues
- Isolated scanner-runner service
- Semgrep, Gitleaks and OSV adapters
- Persistent scan history and comparison
- Streaming scan logs with WebSocket/SSE
- Project and workspace CRUD

## Phase 3 — Asset discovery and attack graph
- AST-based endpoint and auth discovery
- Docker, CI/CD and database asset extraction
- Relationship graph construction
- Graph traversal and attack-path scoring
- Interactive 2D graph filters and evidence drawer
- Finding deduplication and lifecycle management

## Phase 4 — Remediation and local AI
- Remediation workflows, assignments and comments
- Browser-local embeddings for duplicate grouping
- Optional Ollama patch/test generation
- Regression rule generation
- Verification scans and risk acceptance

## Phase 5 — GitHub and release security
- GitHub App integration and signed webhooks
- Pull-request differential scanning
- GitHub Checks annotations
- Release readiness scoring
- Scheduled rescans and notifications

## Phase 6 — Simulation and 3D intelligence
- Full 3D security mesh driven by real graph data
- Safe attack-path simulation playback
- Security time machine
- Architecture drift detection
- WebGL performance modes and accessibility fallback

## Phase 7 — Production hardening
- Multi-tenant organizations and RBAC
- Rate limiting, CSRF and strict authorization tests
- Scanner sandbox limits and network isolation
- OpenTelemetry, structured logs and health checks
- Playwright E2E, integration and security test suites
- CodeQL, dependency updates and release automation

## Phase 8 — Public growth and SEO/GEO
- Public sanitized security reports
- CWE/OWASP knowledge pages
- Technical benchmark and methodology pages
- Structured data and multilingual content
- Documentation portal and plugin SDK
