# Phase 25 — Kubernetes Security

Adds deterministic Kubernetes workload security analysis.

Included:
- privileged container detection
- root and privilege-escalation checks
- host namespace sharing detection
- dangerous Linux capability detection
- read-only root filesystem control
- service-account token control
- NetworkPolicy coverage
- CPU and memory limit checks
- immutable image pinning
- deployment blocking decision
- unit tests
- root verification integration

Workload snapshots can be supplied as local JSON.
No cluster connection, API key, subscription or paid service is required.
