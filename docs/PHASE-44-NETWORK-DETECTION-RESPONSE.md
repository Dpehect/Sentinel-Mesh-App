# Phase 44 — Network Detection and Response

Adds deterministic NDR analysis from local network-flow data.

Included:
- lateral movement detection
- SMB, RDP and remote administration analysis
- periodic beaconing detection
- malicious destination and C2 matching
- DNS tunneling detection
- large outbound transfer detection
- internal scanning detection
- kill-chain stage classification
- automatic source-isolation decision
- unit tests
- root verification integration

Flow records can be supplied as local JSON or imported sensor output.
No packet-intelligence API, subscription or paid service is required.
