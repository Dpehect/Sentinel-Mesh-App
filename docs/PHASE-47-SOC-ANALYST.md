# Phase 47 — SOC Analyst

Adds deterministic, evidence-grounded SOC triage.

Included:
- multi-source alert correlation
- severity and confidence aggregation
- timeline generation
- evidence deduplication
- MITRE ATT&CK aggregation
- recommended containment and investigation actions
- human-approval flags for destructive actions
- close, investigate and contain decisions
- compact analyst brief
- unit tests
- root verification integration

This module does not require an external AI model.
It can later serve as the deterministic safety layer for a local LLM.
No API key, subscription or paid service is required.
