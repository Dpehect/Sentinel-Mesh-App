# Phase 46 — UEBA 2.0

Adds advanced deterministic identity and entity behavior analytics.

Included:
- peer-group behavior comparison
- impossible-travel detection
- dormant account reactivation
- privileged dormant identity risk
- authentication-to-privilege sequence correlation
- abnormal data-volume detection
- rapid multi-country behavior
- high-confidence entity containment decisions
- unit tests
- root verification integration

Behavior profiles, events and peer metrics can be supplied as local JSON.
No machine-learning API, subscription or paid service is required.
