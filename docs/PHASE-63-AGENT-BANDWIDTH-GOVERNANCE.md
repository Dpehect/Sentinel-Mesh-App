# Phase 63 — Agent Bandwidth Governance

Adds deterministic transfer-budget and backpressure controls.

Included:
- per-agent bandwidth windows
- maximum batch sizes
- critical telemetry reservation
- priority-based transfer planning
- gzip and Brotli size estimation
- compression thresholds
- expired-item removal
- deferred transfer generation
- send, throttle and block decisions
- unit tests
- root verification integration

No external traffic-management or paid service is required.
