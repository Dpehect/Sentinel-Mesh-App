# Phase 32 — UEBA

Adds deterministic User and Entity Behavior Analytics.

Included:
- user, service-account, device and workload entities
- usual-hour baseline checks
- country and source-location anomalies
- unusual action detection
- abnormal data-transfer detection
- privileged failure detection
- abnormal event-volume detection
- allow, investigate and contain decisions
- unit tests
- root verification integration

Phase 32 also repairs Phase 30 and Phase 31 package integration.
No AI API, subscription or paid service is required.
