# Phase 27 — CI/CD Security

Adds deterministic pipeline security analysis.

Included:
- dangerous pull_request_target detection
- protected branch enforcement
- production approval checks
- immutable action version checks
- excessive workflow permission detection
- secret exposure detection
- remote download-and-execute detection
- security gate bypass detection
- merge/deploy decision
- unit tests
- root verification integration

Pipeline definitions can be supplied as local JSON.
No CI provider API key, subscription or paid service is required.
