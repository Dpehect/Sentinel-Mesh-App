# Phase 1 — Enterprise Baseline

This phase adds the minimum governance and CI baseline expected from an enterprise security product.

## Included
- security reporting policy
- threat model
- production security checklist
- free GitHub Actions verification
- free Dependabot configuration
- pull request template

## Cost
No paid API, external account or API key is required.

## Verification
The workflow runs:
1. dependency installation
2. static verification
3. type checking
4. unit tests
5. production build

The existing `npm run verify` command remains the source of truth.
