# Phase 40 — External Attack Surface Management

Adds deterministic external asset inventory and exposure monitoring.

Included:
- domains, subdomains, IPs, certificates, services and storage endpoints
- owned, third-party and unknown ownership states
- public admin and login exposure detection
- dangerous internet-facing port detection
- TLS and certificate-expiry checks
- public write-access detection
- newly discovered and removed asset changes
- risk increase detection
- monitor, investigate and urgent decisions
- unit tests
- root verification integration

External observations can be supplied as local JSON or imported scanner output.
No DNS intelligence API, subscription or paid service is required.
