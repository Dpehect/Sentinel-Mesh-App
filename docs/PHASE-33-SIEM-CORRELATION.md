# Phase 33 — SIEM Correlation

Adds deterministic multi-source security-event correlation.

Included:
- normalized security event model
- time-window correlation
- entity, asset, IP and tenant grouping
- multi-source rule requirements
- event sequence validation
- incident confidence scoring
- automatic incident deduplication
- allow, investigate and escalate decisions
- unit tests
- root verification integration

Events and correlation rules can be supplied as local JSON.
No SIEM subscription, API key or paid service is required.
