- [ ] CI, CodeQL and dependency review passing
- [ ] E2E tests passing on desktop and mobile
- [ ] No default credentials in production
- [ ] Threat model and security policy reviewed
- [ ] Public report publication authorization tested
- [ ] Scanner versions displayed in reports
- [ ] Benchmark results reproduced on documented hardware
- [ ] LICENSE, CONTRIBUTING and CODE_OF_CONDUCT present
- [ ] v1.0.0 release notes published
