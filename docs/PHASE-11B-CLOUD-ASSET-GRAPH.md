# Phase 11B — Cloud Asset Graph

Adds deterministic topology and attack-path correlation on top of Phase 11A.

## Included
- Provider-independent graph node and edge model
- Internet exposure correlation
- Kubernetes service → workload → identity relationships
- Prioritized attack-path discovery
- Interactive graph UI
- Authenticated graph API
- PostgreSQL graph persistence schema
- Unit tests

No cloud login, API key, paid service or external AI is required.
