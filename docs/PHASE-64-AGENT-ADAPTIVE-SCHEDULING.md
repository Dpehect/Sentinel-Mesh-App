# Phase 64 — Agent Adaptive Scheduling

Adds deterministic resource-aware agent task scheduling.

Included:
- CPU and memory budgets
- battery and external-power awareness
- active-user protection
- metered-network restrictions
- maintenance windows
- critical-task overrides
- concurrency limits
- running-task suspension under resource pressure
- schedule, defer and suspend decisions
- unit tests
- root verification integration

No external scheduler or paid service is required.
