# Apply Phase 11A

1. Extract this ZIP.
2. Copy all folders and files into the root of `Sentinel-Mesh-App`.
3. Allow replacement of existing `package.json`, `apps/web/package.json`, and `apps/web/src/lib/navigation.ts`.
4. Run:

```bash
npm install
npm run verify
```

For persistent mode also run:

```bash
npm run db:migrate
```

Then open `/posture`.

No API key, paid service, cloud account, or registration is required.
